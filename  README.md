# Desafio FSBR - Backend (API Cadastramento)

API REST desenvolvida em **Java + Spring Boot** para o desafio FSBR.

O sistema possui funcionalidades de:
- Inclusão de Pessoa
- Alteração de Pessoa
- Exclusão de Pessoa
- Consulta por CPF
- Listagem por Nome
- Listagem Geral
- Autenticação via JWT (Bearer Token)
- Listagem de Estados via endpoint

---

## 🛠 Tecnologias Utilizadas
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security + JWT
- MySQL
- Maven

---

## 📋 Requisitos para executar
- Java 21+
- Maven 3.8+
- MySQL 8+
