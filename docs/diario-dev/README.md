# Diário de Desenvolvimento

Esta pasta registra a evolução do Projeto Fiscal em entradas curtas e cronológicas. O diário evidencia o processo de engenharia, incluindo objetivos, entregas, dificuldades, validações e próximos passos.

Para criar uma entrada, copie [`TEMPLATE.md`](TEMPLATE.md), renomeie o arquivo conforme a convenção `AAAA-MM-DD.md` e substitua as orientações pelo conteúdo do dia. Quando houver mais de uma entrada na mesma data, acrescente o tema ao nome do arquivo.
