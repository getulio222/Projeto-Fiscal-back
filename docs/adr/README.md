# Registros de Decisão Arquitetural

Esta pasta contém os ADRs (*Architecture Decision Records*) do projeto. Cada registro documenta uma decisão relevante, seu contexto, as alternativas consideradas e os impactos esperados.

## Como utilizar

1. Copie o arquivo [`TEMPLATE.md`](TEMPLATE.md).
2. Renomeie a cópia seguindo o padrão `NNNN-titulo-da-decisao.md`.
3. Preencha todas as seções aplicáveis antes de aceitar a decisão.
4. Atualize o status quando a decisão for aceita, rejeitada, descontinuada ou substituída.
5. Caso outro ADR substitua a decisão, mantenha ambos e inclua referências cruzadas.

Os números devem ser sequenciais e não devem ser reutilizados.
