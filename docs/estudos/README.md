# Estudos Técnicos

Esta pasta reúne pesquisas, experimentos e análises usados para desenvolver competências ou fundamentar escolhas do Projeto Fiscal. Os documentos devem distinguir fatos, hipóteses e conclusões, além de indicar fontes confiáveis.

Para iniciar um estudo, copie [`TEMPLATE.md`](TEMPLATE.md), renomeie o arquivo conforme o padrão `AAAA-MM-DD-tema.md` e adapte as seções à investigação realizada.
